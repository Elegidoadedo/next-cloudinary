# Optimizing Images in Next.js

Automatically optimize images using the CldImage component. By default, CldImage opts you in to automatic optimization including delivering the most optimal format for the browser (WebP, AVIF).

You can further optimize delivery by using [responsive sizing](/guides/responsive-images) by using the `sizes` prop.

<Video
  title="Optimize Images, Responsive Sizing, & AI Cropping in Next.js with Next Cloudinary"
  url="https://www.youtube.com/watch?v=82gACPNBkaM"
/>

## Example

<CldImage
  width="960"
  height="600"
  src={`${process.env.IMAGES_DIRECTORY}/turtle`}
  sizes="100vw"
  alt="Turtle"
/>

```jsx
import { CldImage } from 'next-cloudinary';

<CldImage
  width="960"
  height="600"
  src="images/turtle"
  sizes="100vw"
  alt="Turtle"
/>
```
